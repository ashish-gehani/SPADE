# How to add a make-based build to a submodule

Use `module/android` as the reference implementation.

## Files to create

### `configure.ac`

Only `configure.ac` is created by hand. Specify the autoconf prerequisite version with `AC_PREREQ` and the automake version inside `AM_INIT_AUTOMAKE`. After any change to `configure.ac` or `Makefile.am`, regenerate the build system with:

```
autoreconf -fi
```

### `Makefile.am`

Define these targets:

- `all-local` — build everything into a local directory (do not install anywhere)
- `install-exec-local` — copy built artifacts out using `$(DESTDIR)$(prefix)` so the caller controls the destination
- `uninstall-local` — reverse of `install-exec-local`
- `clean-local` — remove build artifacts
- `distclean-local` — call `clean-local`
- `maintainer-clean-local` — call `distclean-local`, then remove generated autotools files (`configure`, `Makefile.in`, `aclocal.m4`, `build-aux`, `autom4te.cache`)

By default all build output goes into a local subdirectory (e.g. `build/`). The `install-exec-local` target is the only way to move files out; callers pass `--prefix` and `DESTDIR` to `./configure` / `make install` to place them wherever needed.

### `bin/`

Any helper scripts go in `bin/`. Keep executable scripts there; `Makefile.am` should reference them via that path.

### `.gitignore`

Start from android's `.gitignore` and add any module-specific entries on top:

```
target/
build/
Makefile
config.log
config.status
configure~
autom4te.cache
install
```

## Autogenerated files — do not consult

The following files and directories are produced by `autoreconf -fi` and should never be read or edited directly:

- `autom4te.cache/`
- `build-aux/`
- `aclocal.m4`
- `configure`
- `configure~`
- `Makefile.in`

The authoritative sources are `configure.ac` and `Makefile.am`.

## Workflow

```
# First-time setup (or after editing configure.ac / Makefile.am)
autoreconf -fi

# Configure — all output stays local by default
./configure

# Build locally
make

# Install to a specific location
make install prefix=/some/path DESTDIR=/some/destdir

# Clean up
make clean          # remove build artifacts
make distclean      # clean + remove configure-generated files
make maintainer-clean  # distclean + remove autotools-generated files
```
