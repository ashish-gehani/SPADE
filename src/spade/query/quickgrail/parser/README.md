# QuickGrail Parser

The following files are generated by ANTLR 4 from `DSL.g4` and must be regenerated whenever the grammar changes:

- `DSLLexer.java`
- `DSLParser.java`
- `DSLListener.java`
- `DSLBaseListener.java`
- `DSL.tokens`
- `DSLLexer.tokens`
- `DSL.interp`
- `DSLLexer.interp`

## How to Regenerate

**1. Download the ANTLR 4.13.2 tool jar** (one-time):

```bash
curl -O https://www.antlr.org/download/antlr-4.13.2-complete.jar
```

**2. Run the ANTLR tool on `DSL.g4`** from the repo root:

```bash
java -jar antlr-4.13.2-complete.jar \
  -package spade.query.quickgrail.parser \
  -o src/spade/query/quickgrail/parser \
  src/spade/query/quickgrail/parser/DSL.g4
```

The generated files will be written into this directory, overwriting the previous versions.
